# CoreDefance

CoreDefance is a lightweight cybersecurity project designed to detect suspicious and malicious files in real time. It uses file hashing and pattern-based analysis to identify potential threats quickly and efficiently.

---

## Features

- Real-time file scanning
- SHA-256 hash analysis
- Malware blacklist checking
- Suspicious pattern detection
- Fast and lightweight performance

---

## How It Works

1. The system scans the selected file.
2. It calculates the SHA-256 hash value.
3. The hash is compared with known malicious signatures.
4. The file is analyzed for suspicious behavior or patterns.
5. The final result is shown as Safe or Suspicious.
